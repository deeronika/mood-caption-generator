# 🌙 Mood Caption Generator

A simple Flask web app that creates Instagram-style captions based on your mood.

---

## ✨ Features
- Enter one or multiple moods (sad, happy, crazy, soft, etc.)
- Generates random captions
- Clean and minimal UI

---

## 🚀 Tech Stack
- Python (Flask)
- HTML
- CSS
- JavaScript

---

## 💡 Example
Input:
sad, soft

Output:
"Be gentle with yourself today 🌙"

---

## ▶️ Run Locally

```bash
python app.py
